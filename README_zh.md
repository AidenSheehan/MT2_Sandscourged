## 概述

***在地狱边缘，一股奇异的在天堂前所未见的力量正在显现……***

沙蚀氏族是一个功能全面的氏族，有两名勇者，一整套卡牌和神器，以及一些新机制！其风格与原版游戏相仿。

沙蚀氏族的核心设计包括：

* 祸患和天灾
* 牺牲单位
* 施加减益
* 弃牌

本模组引入两种新减益：巫术（[自制效果与关键词的库](https://github.com/Monster-Train-2-Modding-Group/Conductor)模组的改版）和扭曲，以及一种新触发能力：被咒。特别感谢 [Brandon](https://github.com/TricksterGuy) 制作了[自制效果与关键词的库](https://github.com/Monster-Train-2-Modding-Group/Conductor)模组帮助我！

## 安装
建议使用模组管理器来安装本模组，因为它会自动安装好所有前置。或者，执行下面的手动安装说明。

### 前置
确保在使用模组之前安装了所有的前置。本模组需要：

* [BepInEx-BepInExPack](https://github.com/risk-of-thunder/BepInEx)
* [MT2-Trainworks_Reloaded](https://github.com/Monster-Train-2-Modding-Group/Trainworks-Reloaded)
* [自制效果与关键词的库](https://github.com/Monster-Train-2-Modding-Group/Conductor)

对于每个前置，请按照每个前置自己的安装说明进行安装。

### 本模组
安装所有前置之后，将模组内容（解压缩后）复制到 plugins 文件夹。该文件夹的位置在类似于 `C:\Program Files (x86)\Steam\steamapps\common\Monster Train 2\BepInEx\plugins` 的地方。

## Bug 和反馈
本模组仍然在测试中，因此，您可能会碰到一些 Bug。如果您碰到了，请点击[这里](https://github.com/AidenSheehan/MT2_Sandscourged/issues)汇报。

如果您想要对本模组提供反馈，那就请不要犹豫地提供它吧。请在[这个谷歌表单](https://forms.gle/fL7YDzpHRxnQ9mtA8)填写反馈。

如果有任何问题，可以在[怪物火车 2 的 Discord 服务器](https://discord.gg/jfbTdktH)里联系我(@All_Outta_Bubblegum)。

### 已知 Bug
* 列车日志的氏族进度记录显示在屏幕之外。该 Bug 在官方出 DLC 之前不会修复。
* 单位贴图在战斗中过暗或穿模。这个是模组框架的 Bug。
* 单位战旗的贴图没有正确加载。这个也是模组框架的 Bug。
* 打开列车日志时有极小概率会产生软锁。目前还不清楚该 Bug 的成因，所以如果您碰到了，请将日志文件发送给我。
